ФУНКЦИЯ sum( ДВУМЕРНЫЙ МАССИВ matrix )
| УКАЗАТЕЛЬ НА ЦЕЛОЧИСЛЕННЫЙ МАССИВ:
| sum
| 
| ЦЕЛОЧИСЛЕННЫЕ:
| size_w = matrix[0][0]
| size_h = matrix[0][1]
| 
| ЗАДАТЬ РАЗМЕР size_w ДЛЯ МАССИВА sum_el
|	ДЛЯ i = 1 ПОКА i<=size_w ДЕЛАТЬ i++
|	| sum_el[i-1] = 0
|	|	ДЛЯ j = 0 ПОКА j < size_h ДЕЛАТЬ j++
|	|	| sum_el[i - 1] = sum_el[i - 1] + matrix[i][j]
|	|	КОНЕЦ
|	| ПЕЧАТЬ В КОНСОЛЬ sum_el[i - 1]
|	КОНЕЦ
| ВЕРНУТЬ МАССИВ sum_el
